## Building and Running

### Prerequisites
- Java 17 or higher
- Maven 3.6+

### Build
```bash
mvn clean compile
```

### Run
```bash
mvn spring-boot:run
```

The application will be available at: http://localhost:8080/simplegreeting

### Package
```bash
mvn clean package
```

This creates an executable JAR in the `target/` directory.
