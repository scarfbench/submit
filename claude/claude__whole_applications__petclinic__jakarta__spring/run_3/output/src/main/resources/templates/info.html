About the Petclinic Application

This application has been migrated from Jakarta EE 10 to Spring Boot 3.

It demonstrates a typical CRUD application with the following entities:

  • Owners - Pet owners with contact information
  • Pets - Pets belonging to owners
  • Pet Types - Types/breeds of pets
  • Visits - Veterinary visits for pets
  • Veterinarians - Vets with specialties
  • Specialties - Medical specialties of vets
